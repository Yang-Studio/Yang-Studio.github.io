visualize Build system

This is a visual construction system built using the Unreal Engine Blueprint system, allowing players to freely place and preview building components within the scene, enabling a construction process similar to that found in games like Rust and ARK. I have modularized the system and integrated dynamic materials and compositing to facilitate debugging and expansion.

Survival System

Players lose hunger and dehydration points every 3 seconds while on the map. Players must eat meat or berries to replenish these points. When hunger and dehydration points reach zero, players will begin to lose health.

Craft & Equip System

Players can use collected items to synthesize buildings, tools, or equipment.

Explore Map

Maps and mini-maps will help players explore the map more effectively. The entire map is an empty island with numerous special buildings that have unique functions.

AI NPC

AI NPCs will be generated based on the player's movements. The AI will switch between three states: attacking the player, chasing the player, and roaming. When it sees the player, it will chase and attack them.

Procedurally Generated Map

  • The instance landscapes on the map are generated using a Procedural Content Generation Framework.
  • The map is identified using 64 Zone blueprint actors. If an instance mesh is hit by a player, the Zone blueprint converts the instance mesh into a destructible blueprint. If the player leaves the current Zone blueprint, a random instance mesh is regenerated after 15 seconds. Additionally, AI creatures are generated when the player enters the Zone blueprint.
