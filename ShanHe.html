Core Features

Execution System

When an enemy's health is low enough to be killed, they enter an executable state, triggering a QTE mechanism. The system automatically controls camera lock, character movement, and animation playback, providing a smooth execution experience.

Lock-On System

Target locking is achieved through linear detection, supporting smooth switching between multiple targets. The camera intelligently tracks targets, providing a precise combat experience.

Mission System

A data-driven mission system that supports various mission types including collection, elimination, and location-based objectives. Features a complete mission panel, tracking, dialogue, and navigation functionality.

Weapon Wheel System

A circular weapon selection menu implemented with custom Widgets, using material nodes to create interactive highlighting effects, providing an intuitive weapon switching experience.

Combo System

A structured combo management system supporting light and heavy attack combinations. Smooth combo transitions are achieved through time window judgments, with each weapon having a unique combo table.

Hit Feedback System

Integrates frame freezing, camera shake, hit effects, and impact animations to create a clear and powerful hit sensation.
